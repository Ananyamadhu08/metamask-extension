# ModalContent

The `ModalContent` is a transparent overlay that covers the entire screen. It is used to dim the background when a modal is open.

<Canvas>
  <Story id="components-componentlibrary-helptext--default-story" />
</Canvas>

## Props

The `ModalContent` accepts all props below as well as all [Box](/docs/components-ui-box--default-story#props) component props

<ArgsTable of={ModalContent} />

### Children

Use the `children` prop to render the content of `ModalContent`

<Canvas>
  <Story id="components-componentlibrary-modalcontent--children" />
</Canvas>
```jsx import {(ModalContent, Text)} from '../../component-library';

<ModalContent>
  <Text>Modal Content</Text>
</ModalContent>
```

### Size

Use the `size` prop and `ModalContentSize` enum to set the size of the `ModalContent`
Defaults to `ModalContentSize.Sm`

<Canvas>
  <Story id="components-componentlibrary-modalcontent--size" />
</Canvas>

```jsx
import { ModalContent, ModalContentSize, Text } from '../../component-library';

<ModalContent size={ModalContentSize.Sm}>
  <Text>Modal Content</Text>
</ModalContent>
<ModalContent size={ModalContentSize.Md}>
  <Text>Modal Content</Text>
</ModalContent>
<ModalContent size={ModalContentSize.Lg}>
  <Text>Modal Content</Text>
</ModalContent>
```
